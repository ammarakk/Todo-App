FastAPI backend for Todo SaaS application with Hugging Face AI integration.

## Features
- FastAPI REST API
- JWT Authentication
- PostgreSQL Database
- Hugging Face AI Integration
- Todo CRUD Operations

## API Endpoints
- Health: `/health`
- Auth: `/api/auth/*`
- Todos: `/api/todos/*`
- Users: `/api/users/*`
- AI: `/api/ai/*`
